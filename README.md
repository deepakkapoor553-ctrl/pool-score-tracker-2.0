# 🎱 8-Ball Pool Score Tracker

A simple, mobile-friendly web application to track scores for 8-ball pool races. Add players, adjust scores with one tap, and see live standings – all data is saved locally in your browser.

## ✨ Features

- ➕ **Add/Remove Players** – quickly add players with names and delete them anytime.
- 🔢 **Score Adjustment** – increase or decrease scores using **+2 to +7** and **+10** buttons.
- 🔻 **Negative Scores Allowed** – scores can go below zero, displayed in red for clarity.
- 🏆 **Live Standings** – automatically ranks players by total points.
- 📊 **Real-time Stats** – shows total players, total points, current leader, and highest score.
- 💾 **Local Storage** – all data is saved in your browser using IndexedDB – no server required.
- 📱 **Mobile Friendly** – fully responsive design that works on phones, tablets, and desktops.

## 🎯 How to Use

1. **Add a Player** – type a name in the input box and click **"Add"**.
2. **Adjust Score** – click any **+** button to increase points, or any **−** button to decrease points.
3. **View Standings** – the table on the right shows all players ranked by points.
4. **Remove a Player** – click the 🗑️ icon next to any player to delete them (confirmation required).
5. **Reset All** – click the **"Reset"** button in the header to delete all players and scores.
